# 보류한 기능·아이디어

- **상태**: 보류

## 줄 끝 기호 표시

- **아이디어**: 개행 위치에 줄 끝 기호(¶ 등)를 보여 주기.
- **판단**: Win32 기본 `EDIT`만으로는 자연스럽게 구현하기 어렵다.
- **후속**: RichEdit, Scintilla, 커스텀 렌더링 검토 시 재논의. (`doc/upcoming-release-note.md`에 기록)

## 시스템 폰트 설정 UI

- **아이디어**: 입력창 폰트를 맑은 고딕 등 사용자가 고르게.
- **판단**: 입력·조합 안정화를 우선하고 UI 외관은 후순위.
- **관계**: 한글 표시 안정성은 `04-embedded-korean-font.md`의 내장 폰트로 일부 대체됨.

## 전역 물리키 훅

- **초기 상세설계**: 1차 릴리스 Out-of-Scope.
- **현재**: 입력창 포커스 안에서만 물리키 변환. 다른 앱 전역 입력은 범위 밖.
